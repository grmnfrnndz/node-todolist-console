# NODE => TODO List

Es una simple aplicacion de listado de tareas.

```
================================================================
                             Menu                               
================================================================

? Que desea hacer...? (Use arrow keys)
❯ 1. Crear una tarea 
  2. Listar tareas 
  3. Listar tareas completadas 
  4. Listar tareas pendientes 
  5. Completar tarea(s) 
  6. Borrar tarea(s) 
  0. Salir 
```
